<h2>Installation</h2>

  Repository requires <a href="https://www.python.org/">Python </a> v3 to run.
  
  <h3>Linux</h3>
  
  ```
  $ git clone https://github.com/rebl0x3r/3xEncoder_Deobfuscator.git
  
  $ cd URLer

  $ chmod +x decoder.py
  
  $ pip3 install colorama

  $ ./decoder.py
  ```
  Or just type :
  ```
  $ python3 decoder.py
   
  ```
